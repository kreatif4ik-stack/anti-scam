# Анти-скам тренажёр

Помогает распознавать мошеннические схемы и проверять свою бдительность.

## Как это работает

Вы проходите интерактивные сценарии и узнаёте, на каком шаге вас могли бы обмануть. Каждый сценарий — это реальная схема мошенников: от звонка «майора ФСБ» до поддельных голосований в Telegram.

## Для кого

- Для тех, кто хочет проверить себя
- Для тех, кого только что обманули — есть экстренная помощь
- Для родственников, которые часто попадаются на уловки

## Источники

Схемы основаны на реальных данных:
- [Сервис АнтиМошенник (САМ)](https://sam.mos.ru/istorii)
- [Газпромбанк — актуальные схемы](https://www.gazprombank.ru/personal/page/schemes/)

## Технологии

- Next.js 14
- Tailwind CSS
- Статический сайт, без баз данных и серверов
